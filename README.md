**Curso:** Programación en Móviles

## Descripción
Proyecto desarrollado en Jetpack Compose que implementa:
- Pantalla de bienvenida
- Exploración de 8 componentes básicos (TextField, Button, Image, Row, Column, Card, Checkbox, RadioButton)
- Tarjeta de presentación personal
- Tarjeta mejorada con interactividad (cambio de tema)

## Tecnologías
- Kotlin
- Jetpack Compose
- Material Design 3
- Android Studio
